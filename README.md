# Library
City Central Library is a public library managing a vast collection of books and thousands of members.
The library manages key operational entities such as books, members, loans, reservations, authors, publishers, and fines.
•
Books are classified by Genre and ISBN. Multiple copies of the same book may exist.
•
Members can borrow books based on their membership tier (Gold, Silver, Basic).
•
Loans track which book is issued to whom, the issue date, and the due date.
•
Fines are calculated automatically if a book is returned after the due date ($1 per day).
•
Reservations allow members to hold a book if all copies are currently checked out.
•
Authors/Publishers are linked to books for cataloging.
